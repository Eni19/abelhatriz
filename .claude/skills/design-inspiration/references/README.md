# References

Drop your design inspiration here: screenshots, mockups, exported Figma frames, competitor UI captures, anything that shows the look and feel you want Claude to mimic.

Tips:
- PNG/JPG/etc. all work fine.
- Descriptive filenames help Claude know what each image is for, e.g. `landing-hero.png`, `dashboard-dark-mode.png`, `pricing-cards.png`.
- If a reference only applies to a specific part of the product, say so in the filename or mention it when you ask Claude to do the related work.
- No images here yet? Claude will proceed using its own judgment until you add some.
